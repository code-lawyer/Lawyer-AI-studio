交互说明

  • 拖拽排序 — 中间区域的步骤卡片支持上下拖拽调整顺序
  • 拖入添加 — 从右侧 Agent 面板拖入 Agent 到流程中插入新步骤
  • 删除步骤 — 点击步骤卡片右侧的 ✕ 移除该步骤
  • 编辑标签 — 点击步骤的描述文字可直接编辑（如"分析起诉状"→"分析合同纠纷"）
  • 预设保护 — 系统预设工作流只能"复制为自定义"后再修改，不能直接编辑
  • 新建工作流 — 左下角按钮创建空白工作流，填写名称和描述后开始添加步骤
