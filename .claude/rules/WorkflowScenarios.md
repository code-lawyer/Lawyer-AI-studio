# 场景识别规则

**版本**: 1.0
**最后更新**: 2026-01-01
**说明**: 定义SuitAgent系统的自动识别规则和场景映射机制，支持8种文档类型的智能识别和对应工作流推荐

本文件定义了SuitAgent系统的自动识别规则和场景映射机制，支持8种文档类型的智能识别和对应工作流推荐。

## 自动识别规则

### 1. 起诉状识别

**文件类型**：起诉状
**识别关键词**：原告、被告、诉讼请求
**自动场景**：被告应诉
**推荐工作流**：DocAnalyzer → IssueIdentifier → Researcher → Strategist → Writer → Reporter
**优先级**：高

### 2. 证据材料识别

**文件类型**：证据材料
**识别关键词**：证据、附件、证明
**自动场景**：证据质证
**推荐工作流**：DocAnalyzer + EvidenceAnalyzer → Researcher → Writer → Summarizer
**优先级**：高

### 3. 庭审笔录识别

**文件类型**：庭审笔录
**识别关键词**：庭审、审判员、当事人
**自动场景**：庭审后分析
**推荐工作流**：DocAnalyzer(庭审笔录) → EvidenceAnalyzer(对比) → Strategist(调整策略) → Summarizer → Reporter
**优先级**：中

### 4. 判决书识别

**文件类型**：判决书
**识别关键词**：判决、裁定、法院认为
**自动场景**：判决分析
**推荐工作流**：DocAnalyzer → IssueIdentifier → Researcher(上诉可行性) → Strategist → Summarizer
**优先级**：中

### 5. 诉前沟通记录识别

**文件类型**：诉前沟通记录
**识别关键词**：委托、咨询、初步沟通、意向
**自动场景**：法律服务方案
**推荐工作流**：DocAnalyzer → IssueIdentifier → Strategist → Writer(服务方案) → Summarizer
**优先级**：中

### 6. 诉讼中沟通识别

**文件类型**：诉讼中沟通
**识别关键词**：客户反馈、对方反应、新情况、补充信息
**自动场景**：策略优化
**推荐工作流**：DocAnalyzer → EvidenceAnalyzer(优化证据) → Strategist(调整策略) → Reporter
**优先级**：中

### 7. 原告案件材料识别

**文件类型**：原告案件材料
**识别关键词**：委托起诉、准备起诉、拟起诉、材料清单
**自动场景**：原告起诉
**推荐工作流**：DocAnalyzer → IssueIdentifier → Researcher → EvidenceAnalyzer → Writer(起诉状) → Writer(证据目录) → Summarizer → Reporter
**优先级**：高

### 8. 委托确定识别

**文件类型**：委托确定
**识别关键词**：确定委托、签署委托、办理委托、委托材料
**自动场景**：制作委托材料
**推荐工作流**：DocAnalyzer → Writer(委托合同) → Writer(授权委托书) → Writer(谈话笔录) → Summarizer → Reporter
**优先级**：高

## 一键启动方式

### 方式一：上传文件自动识别

**描述**：上传文件自动识别

**描述**：描述需求自动分析
**示例**："我想分析这个案件的争议点，并准备答辩"
**流程**：解析需求 → 自动规划流程 → 执行

## 场景优先级定义

2. 证据质证
3. 原告起诉
4. 制作委托材料

## 识别精度要求

- **最低关键词匹配数**：2个
- **模糊匹配**：启用
- **上下文分析**：启用
- **置信度阈值**：0.7

## 特殊情况处理

### 多场景冲突

**处理方式**：按优先级排序，提示用户确认
**优先级顺序**：
1. 被告应诉
2. 证据质证
3. 原告起诉
4. 制作委托材料

### 场景不明确

**处理方式**：请求用户澄清或提供更多信息
**回退机制**：使用通用案件分析流程

### 混合文档

**处理方式**：分别处理各部分内容
**处理原则**：主文档确定主要场景，辅助文档作为补充
